# Contributing to Publix Passport Expense Tracker

Thank you for your interest in contributing to our project! Volunteers and developers like you help make this utility a reliable tool for store associates to manage work-related expenses, uniform costs, and mileage claims.

## Contribution Guidelines
To ensure a smooth collaboration, please make sure your code adheres to standard React best practices, maintains semantic HTML, and keeps styling component-centric. Ensure all new components are backed by descriptive unit tests.

## How to Contribute
1. **Fork the Repository:** Create a personal copy of this repository by clicking the fork button at the top of the page.
2. **Clone Locally:** Clone your fork to your local development machine.
3. **Branching:** Create a new branch with a descriptive name highlighting your changes:
   
```bash
   git checkout -b feature/your-feature-name
