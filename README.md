# Publix Passport Expense Tracker

## Description
The Publix Passport Expense Tracker is an intuitive, specialized financial utility designed specifically for retail store associates and regional management. This application acts as an independent digital logbook, allowing staff members to systematically record, track, and manage out-of-pocket work expenses such as specialty uniform acquisitions, approved store supply purchases, travel reimbursements, and logistical mileage claims. Built to solve the headache of loose paper receipts, the tool organizes all corporate spending entries into an accessible interface, ensuring precision prior to official reimbursement submissions. By bridging the gap between personal outlays and corporate record-keeping, it brings organizational clarity directly to store employees, saving time and reducing accounting discrepancies at the end of the pay cycle. This system guarantees that every dollar spent in service of store upkeep, customer satisfaction, or operational logistics is accounted for, categorized, and ready for validation by store administrative staff.

## Features
- **Dynamic Expense Logging:** Effortlessly record expenses with specific categorizations like Uniforms, Supplies, and Travel.
- **Interactive Receipt Upload Simulator:** Generate digital copies of physical receipts dynamically linked to each item log.
- **Real-Time Financial Dashboard:** View aggregated totals instantly to monitor out-of-pocket metrics.
- **Categorized Data Grid:** Filter and review transaction entries seamlessly for swift bookkeeping.
